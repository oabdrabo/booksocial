Instagram, shaped for reading.

Upload an EPUB and it becomes a post — a cover, a caption, and the book itself, readable right in the feed. Then follow readers, like and save, highlight passages, and start book clubs.

what it does

Your bookshelf is your profile.

EPUBs become readable posts, and reading becomes something you do with people.

EPUB → post

ebooklib parses chapters and paragraphs; HTML is sanitized, covers and images extracted with Pillow.

A real feed

Books from people you follow, with likes, saves, comments, reposts and quotes.

In-app reader

Chapter table of contents, paragraph rendering, reading progress.

Highlights & notes

Mark passages and keep private notes, scoped per book.

Book clubs

Shared spaces around a read, plus DMs and an activity inbox.

Proxy auth

Trusts reverse-proxy identity headers — drops cleanly behind a homelab SSO.

get started

Three commands.

Needs Python. SQLite-backed, single process.

$ git clone https://github.com/oabdrabo/booksocial && cd booksocial
$ pip install -r requirements.txt && python seed.py
$ DEV_MODE=1 flask --app app run   # localhost:5000/?as=demo
